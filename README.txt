# K Dance Manager Web App

정식 웹앱(PWA) 배포용 기본 파일입니다.

## 사용 방법
1. 이 ZIP을 압축 해제합니다.
2. GitHub 저장소에 업로드합니다.
3. Vercel 또는 Netlify에 연결합니다.
4. 배포된 https 링크를 휴대폰 크롬에서 엽니다.
5. 메뉴 ⋮ → 홈 화면에 추가 / 앱 설치를 누릅니다.

## 포함 기능
- 대시보드
- 학생관리
- 상담관리
- 월별 장부 CSV 다운로드
- 문자센터: 문구 생성, 복사, 문자앱 열기
- PWA manifest / service worker
